## 📋 What is ApplyCalleeTypeEx?

ApplyCalleeTypeEx is a small tool that works with IDA Pro, a software used to inspect and analyze other programs. This tool adds a right-click menu so you can quickly set function types. You can also edit function info live and use full support for SAL and MSDN type information.

It works with IDA Pro versions 8.x through 9.3+, making it useful whether you have an older or the newest version.

## 💻 System Requirements

To use ApplyCalleeTypeEx, your computer needs to meet these basic requirements:

- Windows 10 or 11 (64-bit recommended)
- IDA Pro version 8.x or higher installed
- At least 4 GB of free RAM
- Around 50 MB of free disk space for the plugin files
- An internet connection to download the plugin files

No other software or tools are required.

## ⚙️ How to Install the Plugin in IDA Pro

1. Open IDA Pro on your computer.

2. In IDA Pro, locate your `plugins` folder:

   - Usually found in the program directory at `C:\Program Files\IDA Pro\plugins`
   - Or under your user’s config folder, depending on your IDA setup.

3. Copy all files from the extracted ApplyCalleeTypeEx folder.

4. Paste the files into the `plugins` folder.

5. Restart IDA Pro to load the new plugin.

## 🛠️ How to Use ApplyCalleeTypeEx

Once installed:

- Open any program you want to analyze in IDA Pro.
- Right-click on a function in the disassembly view.
- You will see new options from ApplyCalleeTypeEx in the right-click menu.
- Select the function type or prototype you want to apply.
- The live prototype editor allows you to edit function parameters and return types directly in the interface.
- The plugin handles full support for SAL (Standard Annotation Language) and MSDN (Microsoft Developer Network) function annotations, helping you get detailed prototypes and parameter info automatically.

This improves your analysis by making functions easier to understand and document.

## ⚠️ Troubleshooting Tips

- If the plugin menu does not appear, check that you copied the files to the right `plugins` folder.

- Check your Windows permissions. Running IDA Pro as Administrator might help if permissions block the plugin.
